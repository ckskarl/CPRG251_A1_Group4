===========================================================
		    Group 4 - Assignment 1 - Book Manager
===========================================================
The Program will read a series of books from a text file.
After doing so, it will display a menu where the user can
do several actions. These actions include; checking out a book,
finding books by title, display books of a specific type, produce
a list of random books, and saving and exiting the program.

Run the program by running AppDriver, or by using the JAR
File in the project.
